Netflix Clone

Netflix Clone is a mobile application showcasing advanced UI design and aesthetics, built with Flutter and Dart and utilizing Firebase Authentication for user login. While this project doesn't include API integration, it demonstrates the implementation of various UI elements inspired by the Netflix platform.

 Features
- Firebase Authentication: Allows users to securely log in.
- Ultimate UI: Incorporates nearly all learned UI components for an aesthetic user experience.
- Responsive Design: Supports different screen sizes and orientations.

 Technologies Used
- Flutter
- Dart
- Firebase Authentication**
